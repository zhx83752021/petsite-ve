## 🎯 技术架构

### 后端

- **框架**: Vercel Serverless Functions
- **语言**: TypeScript
- **数据库**: PostgreSQL (Vercel Postgres)
- **认证**: JWT
- **特性**:
  - 事务处理
  - 参数验证
  - 错误处理
  - 日志记录
  - CORS 支持

### 前端

- **框架**: Vue 3 + TypeScript
- **UI 组件**: Element Plus
- **路由**: Vue Router
- **状态管理**: Pinia
- **HTTP 客户端**: Axios
- **构建工具**: Vite
