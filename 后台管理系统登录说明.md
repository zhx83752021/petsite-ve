## 🚀 快速开始

### 1. 初始化数据库

```bash

npm run db:init
```

### 2. 创建管理员账号

```bash
npm run create:admin
```

### 3. 启动后端服务

```bash
npm run dev
```

### 4. 启动前端服务

```bash
cd ../frontend
npm run dev
```

## 🎯 使用步骤

1. **登录**

   - 访问 `http://localhost:5173/admin/login`
   - 输入用户名：`admin`
   - 输入密码：`admin123`
   - 点击登录

2. **查看工作台**

   - 登录成功后自动跳转到工作台
   - 可以看到数据统计和快捷入口

3. **管理管理员**

   - 点击左侧菜单 "系统管理" -> "管理员管理"
   - 可以进行管理员的增删改查操作

4. **退出登录**
   - 点击右上角头像
   - 选择"退出登录"

### Q3: 登录时提示"用户名或密码错误"

- 确认用户名是 `admin`（小写）
- 确认密码是 `admin123`
- 确认数据库已正确初始化

### Q4: 后端服务启动失败

- 检查数据库是否已启动
- 检查 `.env.development` 配置是否正确
- 查看控制台错误日志
